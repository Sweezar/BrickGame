Tetris Game Documentation

Controls

ENTER: Start Game

ESC: End Game

Space: Pause Game

Left Arrow: Move Figure Left

Right Arrow: Move Figure Right

Down Arrow: Drop Figure Down

W: Rotate

Game Rules

The game starts at level 1 and can reach up to level 10. With each new level, the difficulty increases - figure start falling faster.

Scoring

Maximum accumulated score is saved after each game.

Snake Game Documentation

Controls

ENTER: Start Game

ESC: End Game

Space: Pause Game

Left Arrow: Move Left

Right Arrow: Move Right

Down Arrow: Drop Down

W:Acceleration

Game Rules

The game starts at level 1 and can go up to level 10. With each new level, the difficulty increases - the snake begins to move faster. The goal of the game is to eat as many apples as possible; with each apple eaten, the length increases.

Scoring

Maximum accumulated score is saved after each game.
